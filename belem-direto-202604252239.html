Resumo para conduzir

Objetivo

Construir um assistente comercial simples

Interface + IA + registro de dados + ação comercial. O aluno precisa ver algo funcionando, não só entender conceitos.

Mensagem central

IA só gera valor quando entra em processo

O fio da aula é: processo estruturado → automação inteligente → IA em ação.

Público

Empresário com pouco tempo

Linguagem simples, exemplos de negócio e checkpoints rápidos. Sem aula técnica longa.

Entrega final

Mini-sistema apresentável

Cada grupo termina com uma solução adaptada a um nicho e um pitch de 2 minutos.

Falas-chave prontas

Abertura

“Hoje você não vai aprender ferramenta. Você vai construir um sistema simples que usa IA para resolver um problema real.”

Processo

“Automatizar processo ruim só faz o erro acontecer mais rápido.”

Dados

“Dado é memória operacional. Sem isso, toda IA vira conversa solta.”

Fechamento

“O próximo salto não é usar mais IA. É colocar IA dentro da rotina do negócio.”

Plano B se der problema

Internet ruim

Trabalhar com prompt + planilha + simulação de fluxo. O conceito continua vivo.

Lovable falhar

Usar formulário Google/Typeform fake ou tela pronta. Foco no raciocínio.

Supabase travar

Usar Google Sheets como banco didático.

Automação não rodar

Mostrar fluxo desenhado + saída manual gerada por IA.
